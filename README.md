# Smart-Expense-Splitter-
A Java-based efficient expense management system that simplifies group expense management by fairly splitting costs and minimizing the number of transactions between users.
This project is developed as part of my Vityarthi Java 'BYOP' submission.

How It Works:- 
The program calculates each user's fair share of total expenses. 
It then determines the balance of each user
to minimize the number of transactions by matching the highest debtor 
with the highest creditor until all balances are settled.
